# Job Listing UI - PRD

## Original Problem Statement
Build a simple Job Listing user interface that allows users to browse, search, and filter job postings.

## User Personas
- Job seekers looking for frontend/tech positions
- Users browsing jobs on desktop and mobile devices

## Core Requirements
- Display job cards with title, company, location, type
- Filter jobs by location
- Filter jobs by job type (Internship/Full-time)
- Search jobs by title using text input
- Basic responsive layout (mobile-friendly)
- Sort jobs alphabetically by title (Bonus)
- Highlight searched keyword in job title (Bonus)
