# Frontend Job Listing UI

## Objective
A simple frontend application that allows users to browse, search, and filter job listings.  
This project demonstrates frontend fundamentals, UI/UX understanding, and basic logic implementation using mock data.

## Features
- Display job listings in card format
- Search jobs by title
- Filter jobs by location
- Filter jobs by job type (Internship / Full-time)
- Sort jobs alphabetically by title
- Highlight searched keywords in job titles
- Responsive and mobile-friendly UI

## Tech Stack
- React.js
- JavaScript (ES6)
- CSS
- Mock JSON data (no backend)

## Job Data Structure
```json
{
  "id": 1,
  "title": "Frontend Intern",
  "company": "Acme Corp",
  "location": "Remote",
  "type": "Internship"
}

Setup and Run Instructions

Install Node.js (v16 or higher recommended)

Create a React app:

npx create-react-app job-listing-ui


Replace the src folder files with the provided code

Start the development server:

npm start


Open your browser and visit:

http://localhost:3000
